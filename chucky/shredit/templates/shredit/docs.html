What is iOTA ?

iOTA is an platform, that helps user with large csv and json file break that files into smaller chunks. This allows for more efficient data management and allows for easier data analysis. Our platform is user friendly and with our basic system requirement you can easily access our platform on your favourite browser.

Getting started

To use the platform you need to be an authenticated user on our end . Our authenticated user can be identified by and benefits from the following;

As an authenticated user you have access to the following features

  • Full access to the platform
  • A personalised dashboard
  • Set your desired chunk file size
  • Access to 2 format (CSV and JSON)
  • Access to our added features - A download link can be generated to share the chunked files and anyone (unauthenticated/ authenticated/ not a user at all of the platform) can access the chunked files to easily download them.
  • Access to our format we would be adding in the near future
  • Allows users to save data and come back to download your splitted file

Unauthenticated users have access to the following features

  • Visiting the platform to view basic information of the site
  • Access to our landing page, documentation page, about us page and our resource page
  • Access to register to gain full access to the platform

How splitting works on the platform

Authenticated user

You receive what you signed up for when you register for full access! No demands from behind the scenes.

Split on the dashboard

When you give iOTA the file to be transformed with a minimal set of settings, it processes the file and sends you a link to retrieve the transformed contents.

Easily share your files

You can effortlessly send your file to your email or a colleague's email on your dashboard.

Download your splited files

You have the option of downloading a selected file or your entire file in a zip format to your device.

About the platform buildup

Technology used -

  • Frontend - HTML, CSS and Javascript for building and designing of the website

  • Backend - Django (Django framework) - to authenticate users,save user data, uploading of files, validating file formats, chunking files, saving user data and progress and download of chunked files

  • Python - To carry out the splitting and returning of the chunked files

  • Database schema - A database schema organizes data into separate entities, determines how to create relationships between organized entities, and how to apply the constraints on the organized data.
  • A concise database schema was designed to organize the data to be obtained from the user into tables and fields showing the relationships of each set of tables with one another.

Why choose iOTA ?

  • iOTA was born out of understanding of user pain points when dealing with large files and empathy for such users. The platform makes splitting files super convenient - in a couple of clicks, get their files chunked and ready for download.
  • iOTA will keep user details safe with excellent data security and encryption - our users can rest easy knowing their data will not be shared or stolen.
  • iOTA saves user progress and they can come back whenever to get their files.
  • No ads - IOTA does not allow this. Enjoy a wonderful experience getting your task done alongside our pleasant interface.
  • Register to have full access - you get what you signed up for! No backdoor demands.
  • Excellent loading speed - you don’t need to worry about lagging due to file size (once within the limit, we got you!)
